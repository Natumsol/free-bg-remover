# 功能特性说明

## 🎨 全新交互流程

应用现在支持完整的图片处理工作流，从选择到编辑，所有操作都在一个流畅的界面中完成。

### 工作流程

```
选择照片 → 实时处理 → 对比查看 → 背景编辑 → 保存导出
```

## ✨ 核心功能

### 1. 📸 智能选择

- **拖放上传** - 直接拖放图片到应用窗口
- **文件选择器** - 通过按钮选择图片
- **批量处理** - 一次处理多张图片
- **格式支持** - JPG、PNG、WEBP

### 2. ⚡ 实时处理视图

当您选择图片后，应用会：

- ✅ 显示原始图片预览
- ✅ 添加半透明遮罩层
- ✅ 显示处理进度（0-100%）
- ✅ 展示 AI 处理动画
- ✅ 实时更新处理状态

**视觉效果：**
- 旋转的加载动画
- 渐变进度条
- 处理百分比显示
- 提示信息（100% 本地处理）

### 3. 🔄 左右滑动对比

处理完成后，您可以：

- **拖动滑块** - 鼠标左右拖动比较原图和抠图结果
- **精准控制** - 滑块位置从 0% 到 100% 可调
- **实时预览** - 即时看到前后对比效果
- **标签显示** - 清晰标注 "Original" 和 "Processed"

**交互特点：**
- 🖱️ 鼠标拖拽
- 👆 触摸屏支持
- 🎯 精准定位
- ⚡ 流畅动画

### 4. 🎨 背景编辑器

完整的背景定制功能：

#### a) 透明背景
- 默认透明背景
- 棋盘格显示透明区域
- 适合 PNG 导出

#### b) 纯色背景
- **预设颜色** - 12 种常用颜色快速选择
  - 白色、黑色
  - 浅灰、深灰
  - 红、橙、绿、蓝
  - 紫、粉、青、橙
- **自定义颜色** - 颜色选择器任意选择
- **一键应用** - 实时预览效果

#### c) 图片背景
- **上传背景图** - 选择任意图片作为背景
- **自适应填充** - 自动填充整个画布
- **预览显示** - 显示当前背景图

### 5. 💾 保存与导出

- **PNG 格式** - 保留透明度
- **高质量输出** - 保持原始分辨率
- **快速保存** - 一键导出
- **文件命名** - 自动添加 "-no-bg" 后缀

### 6. 📚 历史记录

- **自动保存** - 处理过的图片自动记录
- **快速访问** - 查看最近处理的图片
- **缩略图网格** - 可视化浏览
- **容量限制** - 保留最近 50 张

## 💡 使用技巧

1. **快速对比** - 直接拖动滑块到 50% 位置可以看到完整对比
2. **背景编辑** - 处理完成后再选择背景，可以多次尝试不同效果
3. **批量处理** - 使用"Batch Process"一次选择多张图片
4. **历史记录** - 不用担心丢失，最近 50 张都会自动保存
